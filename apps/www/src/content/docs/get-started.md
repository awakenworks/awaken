Use this path if you are new to Awaken and want a working mental model before wiring production features.

## Read in order

1. [First Agent](/tutorials/first-agent/) for the smallest runnable runtime.
2. [First Tool](/tutorials/first-tool/) to understand tool schemas, execution, and state writes.
3. [Build an Agent](/how-to/build-an-agent/) when you want a reusable project baseline.
4. [Tool Trait](/reference/tool-trait/) before writing production tools.

## Leave this path when

- You need more agent capabilities: go to [Build Agents](/build-agents/).
- You need HTTP or frontend integration: go to [Serve & Integrate](/serve-and-integrate/).
- You need persistence or operational controls: go to [State & Storage](/state-and-storage/) or [Operate](/operate/).
